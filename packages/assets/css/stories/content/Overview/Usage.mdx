	</p>
	<p>
		The compressed file, obtained by the method mentioned before or directly
		from the minified file existing in the CSS Framework package, should be
		called using the enqueue method for WordPress, which will add a{' '}
		<Code>&lt;link&gt;</Code> tag into the <Code>&lt;head&gt;</Code> section
		of the plugin.
	</p>
	<Snippet language="php">{dedent`
function myplugin_styles() {
    $plugin_url = plugin_dir_url( __FILE__ );
    wp_enqueue_style( 'style',  $plugin_url . "/css/shared-ui.css");
}\n
add_action( 'admin_print_styles', myplugin_styles );
	`}</Snippet>
