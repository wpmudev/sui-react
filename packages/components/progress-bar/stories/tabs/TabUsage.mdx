    <h3>2. Progress Indicator</h3>
    <p>The progress indicator indicates how much the process has progressed.</p>

    <h3>3. Progress Track</h3>
    <p>The progress track is the static length of the progress bar where the progress indicator runs.</p>

    <h3>4. Value</h3>
    <p>The value of a progress bar can provide detailed information about the process.</p>

    <h3>5. Cancel</h3>
    <p> If the process needs to be terminated, the option to cancel the process will be available. SUI uses a tertiary button to perform a cancel action.</p>

</Section>

<Section title="Types">
	<img src={Types} alt="Progress Bar Types"/>
    <h3 style={{marginTop: 30}}>1. Determinate</h3>
    <p>When a progress can be calculate, determinate progress bar will be used.</p>

    <h3>2. Indeterminate</h3>
    <p>If the process can’t be determined, use an indeterminate progress bar. There won’t be a progress value for the indeterminate progress bar.</p>

</Section>
