<Section title="States">
	<p>Depending upon the message that Score deliver to users, the Score can be displayed in different states.</p>
	<Row>
		<Col>
			<h2>1. Default</h2>
			<p>To display general progress on the score.</p>

			<h2>2. Success</h2>
			<p>Generally display the low risk or safe state as progress. Score value is set from 61 - 100.</p>

			<h2>3. Warning</h2>
			<p>Display a state or risk on progress with this state. Score value is set from 35 - 60.</p>

			<h2>4. Error</h2>
			<p>Error state is to display the high risk and critical situation. Score value is set from 0 - 34. </p>
    	</Col>
    	<Col style={ { display: "flex", alignItems: "center" } }>
    		<img src={States} alt="Score States" />
    	</Col>
    </Row>
</Section>
