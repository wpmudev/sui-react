<Section title="Native list">
	<p>
		By default, an unordered list <Code theme="red">&lt;ul&gt;</Code> and list items <Code theme="red">&lt;li&gt;</Code> behave as a standard bullet list. However, by applying the <Code theme="red">sui-list</Code> class to the <Code theme="red">&lt;ul&gt;</Code> and the <Code theme="red">sui-list__item</Code> class to each <Code theme="red">&lt;li&gt;</Code>, the list will be unstyled, removing default browser-specific styles.
	</p>

<Section title="Flush variation">
	<p>
		When a bordered list is used inside a card or any other component that already has a border, apply the <Code theme="red">sui-list--bordered-flush</Code> modifier along with <Code theme="red">sui-list--bordered</Code> class on <Code theme="red">sui-list</Code> to remove extra borders and maintain a balanced design.
	</p>
